# Roadmap

## Short-term
- Improve parser accuracy for PDF and Excel invoices.
- Add stronger attachment validation and malware checks.
- Expand API error handling and observability.

## Mid-term
- Add more document formats and provider-specific parsers.
- Introduce retry policies and idempotency safeguards.
- Improve dashboard filters and operational visibility.

## Long-term
- Multi-tenant support.
- Role-based access controls.
- Cloud-native deployment profiles.
